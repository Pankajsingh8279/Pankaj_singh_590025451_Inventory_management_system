# Inventory Management System

# Description
1. This project is a terminal-based Inventory Management System written in C.
2. It allows users to manage product records with full CRUD operations using file handling and input validation.

# Features include:

1. Add new products
2. Display all products
3. Search a product by ID
4. Update existing product details
5. Delete a product
6. Save & Load data using text files
7. Validation for:

# Integer input
# Float input
# String input
# Display of:
# Product ID
# Product Name
# Quantity
# Price
# Data persistence using inventory.txt

# Getting Started
1. Requirements
2. C compiler (GCC recommended)
3. Text editor (VS Code, CodeBlocks, Sublime, Notepad++)

# Build Instructions

1. Clone or download the repository
2.  Navigate to the src directory
3.  cd inventory-management-system/src

 # Usage

# After running the program, you will see a menu:

# ===== Inventory Management System =====
1. Add Product
2. Display Products
3. Search Product
4. Update Product
5. Delete Product
6. Save & Exit

# The program allows you to:
1 Add a Product
2. Product ID
3. Product Name
4. Quantity
5. Price

 # Display Products

1. Shows all product records in table format.
2. Search Product
3. Search using a Product ID.
4. Update Product
5. Modify quantity or price.
6. Delete Product
7. Removes a product from the inventory.
8. Save & Exit
9. Stores all data in inventory.txt for future use.
